# Info

Example code is provided for both JavaScript, along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/anime/info/{id}
```

## Path Parameters

| Parameter |  Type  | Description         | Required? | Default |
| :-------: | :----: | ------------------- | :-------: | :-----: |
|    id     | string | The anime info id |    Yes    |  `N/A`  |


## Request Samples

<Tabs items={["JavaScript"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using the example id of "21" (one piece) and the query of "gogoanime"
        const url = "https://api.pewds.vercel.app/anime/info/21";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
