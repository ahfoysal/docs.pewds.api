# Get Anime Episode Streaming Links

Example code is provided for both JavaScript, along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/anime/watch/{episodeId}
```

## Path Parameters

| Parameter |  Type  | Description                    | Required? | Default |
| :-------: | :----: | ------------------------------ | :-------: | :-----: |
| episodeId | string | The ID of the selected episode |    Yes    |   N/A   |

## Request Samples

<Tabs items={["JavaScript"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        /*
        Using the example episode ID of  'spy-x-family-episode-1',
        */
        const url = "https://api.pewds.vercel.app/anime/watch/spy-x-family-episode-1";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
  
</Tabs>
