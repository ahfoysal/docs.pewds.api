# Movie | Trending

## Route Schema (URL)

```
https://api.pewds.vercel.app/movie/trending
```


## Request Samples

        ```js
        import axios from "axios";

        const url = "https://api.pewds.vercel.app/movie/trending";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };
        ```
