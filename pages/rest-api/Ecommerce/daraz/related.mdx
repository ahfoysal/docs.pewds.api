# Fetch Related Product Info

Example code is provided for both JavaScript , along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/daraz/related/:id?shop={seller_id}
```

## Path Parameters

| Parameter |  Type  | Description                                                             | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------------- | :-------: | :-----: |
|    id     | string |  Get related product from this id.  Example: <br></br>`"266816450"`|    Yes    |   N/A   |

## Query Parameters

| Parameter |  Type  | Description                                                             | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------------- | :-------: | :-----: |
|    shop     | string |  Get product from same store with this shop id.  Example: <br></br>`"116936"` |    No    |   N/A   |

## Request Samples

        ```js
        import axios from "axios";

      
        const url = "https://api.pewds.vercel.app/daraz/related/223866897?shop=116936";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };
        ```
