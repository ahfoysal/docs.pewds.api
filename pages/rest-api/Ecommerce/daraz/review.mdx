# Fetch Product Review

Example code is provided for both JavaScript , along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/daraz/getreview/:id?page=1&per_page=5
```

## Path Parameters

| Parameter |  Type  | Description                                                             | Required? | Default |
| :-------: | :----: | ----------------------------------------------------------------------- | :-------: | :-----: |
|    id     | string |  Get review of a product from this id.  Example: <br></br>`"266816450"`   <br></br>|    Yes    |   N/A   |


## Query Parameters

| Parameter |  Type   | Description                               | Required? | Default |
| :-------: | :-----: | ----------------------------------------- | :-------: | :-----: |
|   page    | integer | The page number of results to return.     |    No     |   `1`   |
|  per_page  | integer | The number of results per page to return. |    No     |  `5`   |
