# Fetch All Categories   

 Example code is provided for both JavaScript , along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/bikroy/categories/
```



## Request Samples

<Tabs items={["JavaScript"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Example request using no paramters
        const url = "https://api.pewds.vercel.app/bikroy/categories/";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
 
</Tabs>
