# Search anything from  youtube

Example code is provided for both JavaScript , along with a response schema.

## Route Schema (URL)

```
https://api.pewds.vercel.app/yt/search/{query}
```
## Path Parameters

| Parameter |  Type  | Description                    | Required? | Default |
| :-------: | :----: | ------------------------------ | :-------: | :-----: |
| query | string | The search query; i.e. the title of the <i class="fa fa-video-camera" aria-hidden="true"></i> you are looking for. |    Yes    |   N/A   |



## Request Samples

<Tabs items={["JavaScript"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Example request using no paramters
        const url = "https://api.pewds.vercel.app/yt/search/man city";
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>
